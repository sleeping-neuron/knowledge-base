# 知识库管理系统

个人知识管理工具，支持 Markdown 笔记编辑、AI 辅助写作与知识体系构建、知识图谱可视化。

## 快速开始

```bash
pip install -r requirements.txt
python main.py
```

浏览器打开 http://localhost:8000

## 功能

### 笔记管理
- Markdown 编辑（双栏实时预览 + 代码高亮 + 表格/图片支持）
- 分类目录 + 标签系统
- 全文搜索（SQLite FTS5）
- 批量操作（删除 / 添加标签 / 移动分类 / AI 整理）
- LaTeX 数学公式渲染（MathJax 3，支持 $...$ 行内和 $$...$$ 块级公式）

### 知识图谱
- `/graph` 页面，D3.js 力导向图展示笔记关联
- 共享标签或同分类的笔记自动连线，权重越高连线越粗越亮
- 深色模式亮色连线，浅色模式深色连线，清晰可辨
- 拖拽节点调整位置、滚轮缩放、点击节点打开笔记

### AI 助手

`/ai` 页面提供四合一 AI 工作台：

#### AI 写作
输入话题，AI 自动生成标题 + Markdown 正文 + 分类 + 标签建议，一键保存。

#### 知识体系完善
- 选择分类 → AI 分析知识缺口（识别缺失的重要子主题）
- 生成完善计划（3~8 篇待写笔记大纲）
- 勾选后一键批量生成并保存

#### 笔记整理
- 从首页批量选择笔记跳转，或手动搜索添加
- 可选填写整理指令（如「合并重复内容」）
- AI 交叉比对、去重、合并，输出结构化操作建议（保留/合并/更新/删除）

#### 批量生成
- 输入大主题 → AI 拆解为多个子主题
- 子主题列表可编辑、删除、手动添加
- 一键批量生成全部笔记并保存

#### 其他 AI 能力
- **AI 摘要**：笔记详情页一键生成摘要 + 相关话题推荐
- **AI 润色**：编辑器内一键润色改写 Markdown 内容
- **AI 标签建议**：根据内容自动推荐标签
- 分场景模型选择（9 个场景）各可独立设置 Flash / Pro

### 主题与语言
- 深色/浅色主题切换
- 中文/英文界面切换
- 时间自动转换为用户本地时区显示

## AI 配置

1. 点击顶栏齿轮图标（⚙）打开 AI 设置
2. 填入 DeepSeek API Key（`sk-...`）
3. 选择默认模型：Flash（快速）或 Pro（旗舰）
4. 可分别为各场景指定不同模型

已配置的 Key 不会回显，仅显示「已配置」状态。API Key 保存在本地 `config.json`，不上传第三方。

## 技术栈

- 后端：Python + FastAPI + SQLAlchemy + SQLite
- 前端：HTML + Tailwind CSS + Alpine.js（CDN 零构建）
- 搜索：SQLite FTS5 全文索引
- 图谱：D3.js v7 力导向图
- 公式：MathJax 3 (tex-chtml)
- AI：DeepSeek V4 API（OpenAI 兼容格式）
